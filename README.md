# ムシポケ

こどもが みつけた むしを しゃしんと こえで きろくする、ずかんアプリのプロトタイプです。

## Vercel

Vercelでは `vercel.json` により、トップページ `/` が `outputs/index.html` に向きます。

公開に必要な主なファイル:

- `outputs/index.html`
- `outputs/assets/`
- `vercel.json`

## GitHubに入れるもの

このフォルダをGitHubにアップロードすれば、そのままVercelで公開できます。

入れるもの:

- `outputs/`
- `vercel.json`
- `README.md`
- `.gitignore`

入れないもの:

- `.DS_Store`
- `work/`
- `*.log`
